Website ini memiliki tampilan yang responsif dan modern, dilengkapi dengan navbar, hamburger menu untuk tampilan mobile, serta antarmuka chat yang sederhana dan mudah digunakan. Chatbot pada website ini memungkinkan pengguna untuk bertanya seputar produk make up, rekomendasi kecantikan, serta tips perawatan wajah secara real-time.
Integrasi dengan n8n Webhook memungkinkan website mengirimkan pesan pengguna ke AI Agent dan menerima balasan dalam format JSON, sehingga komunikasi dapat berjalan secara otomatis dan efisien. Project ini dibuat sebagai sarana pembelajaran dalam mengimplementasikan frontend sederhana yang terhubung dengan sistem backend berbasis workflow automation.
